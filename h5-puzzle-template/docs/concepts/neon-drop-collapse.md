# Neon Drop Collapse — 게임 피치

> 쏟아진다. 쌓인다. 같은 색이 3개 이상 닿으면 터진다.

---

## 한 줄 소개

컬럼 상단에서 낙하하는 네온 타일을 좌우 이동으로 착지시키고, 같은 색 3개 이상 인접 시 연쇄 폭발로 점수를 올리는 중력 기반 퍼즐.

---

## 장르 / 레퍼런스

- 장르: 낙하 + 색상 매칭 퍼즐 (Puyo Puyo / Columns 계열)
- 레퍼런스: Puyo Puyo, Columns, Candy Crush (매칭 원리만)
- 차별점: 기존 SameGame의 **`_applyGravity()` + BFS getGroup()** 직접 재사용 — 착지 후 연쇄 폭발이 SameGame 로직과 동일한 구조로 동작.

---

## 핵심 메커닉

1. 8열 그리드, 위에서 타일 1개씩 낙하
2. 플레이어는 **좌/우 버튼**으로 낙하 중 타일의 열 위치 이동
3. 바닥 또는 쌓인 타일에 착지
4. 착지 후 **같은 색 3개 이상 인접**(수평/수직) 시 자동 폭발 제거
5. 제거 후 위 타일이 중력으로 재낙하 → 연쇄 폭발(체인) 가능
6. 타일이 천장에 닿으면 게임 오버

```
착지 전:          착지 후(연쇄):
□ □ □ C □ □ □ □  → C 착지 → C C C 연쇄 제거 → 위 타일 재낙하
□ □ □ □ □ □ □ □     □ □ □ □ □ □ □ □
□ C □ □ □ □ □ □     □ □ □ □ □ □ □ □
□ C C □ R R R □     □ R R R □ □ □ □ (R은 3개 → 폭발!)
```

│                          │
│  □□□□□□□□               │
│  □□□□□□□□               │  ← 8×16 Canvas 그리드
│  ■ ↓ □□□□□□             │     (낙하 타일: 글로우 이동)
│  ■■□□□□□□               │     (위험: 상단 2줄 붉은 펄스)
│  ■■■□■■□□  ← BOOM!     │
├──────────────────────────┤
│      [←]  [→]           │  ← 이동 버튼

- 낙하 타일 고스트: 착지 예정 위치 반투명 아웃라인 (`globalAlpha`)
- 천장 2칸 이내: 상단 붉은 펄스 경고 (CSS keyframes)
- 연쇄 폭발: `ParticleSystem` 600-800개 그대로

## 그래픽 재사용

| 리소스 | 재사용 방법 |
|--------|------------|
| `COLOR_PALETTE` 6색 | 낙하/쌓인 타일 fill/glow 무수정 |
| `ParticleSystem` | 폭발 시 파티클 최대 활용 |
| `FloatText` | 체인 배율 + 점수 팝업 |
| `_applyGravity()` | 착지 후 재낙하 로직 **직접 재사용** |
| `getGroup()` BFS | 인접 동색 그룹 판정 **직접 재사용** |
| 배경 + CSS 토큰 | 무수정 |

신규 그래픽: 고스트 프리뷰 (globalAlpha), NEXT 프리뷰 패널 (CSS div), 위험 경고 keyframe

---

## 개발 난이도

**3 / 5** (자체평가 2/5는 낙관적)  
`_applyGravity()` + `getGroup()` 재사용은 실제 이점이 크나,  
실시간 낙하 루프(`setInterval`) + 착지 판정 + 연쇄 애니메이션 딜레이 큐 관리가 핵심 공수.  
`GameLogic` 상속 불가 → `DropGameLogic` 별도 클래스 작성 필요.

**예상 프로토타입 기간: 1-2주**

---

## 리텐션 훅

- 라운드 클리어 구조: 일정 타일 제거 시 다음 라운드, 낙하 속도 증가
- 연쇄 체인 콤보: Puyo Puyo 계열 검증된 중독 루프
- 리더보드: 최고 라운드 + 최고 점수 이중 랭킹

---

## 리스크

- **Puyo Puyo 유사성**: 메커닉이 잘 알려진 장르라 독창성 소구 어려움 (originality 4.7)
- **캐주얼 UX 허들**: 낙하 속도 + 좌우 이동 조합이 클릭 한 번인 SameGame보다 조작 복잡
- 보완책: 첫 3라운드는 매우 느린 낙하 속도로 튜토리얼 역할
