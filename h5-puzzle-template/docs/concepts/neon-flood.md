# Neon Flood — 게임 피치

> 색상 하나를 골라라. 네온이 퍼진다. 전체를 물들여라.

---

## 한 줄 소개

6색 네온 타일로 뒤덮인 그리드를 **단 하나의 색으로** 점령하는 미니멀 전략 퍼즐.

---

## 장르 / 레퍼런스

- 장르: 색상 점령 퍼즐 (Flood-It 변형)
- 레퍼런스: Flood-It (2010), Inks., Hue (Steam)
- 차별점: 네온 사이버 비주얼 + 데일리 챌린지 + Playgama 리더보드

---

## 핵심 메커닉

1. 14×14 그리드, 6색 타일로 시작
2. 플레이어는 하단 **6색 버튼 중 하나**를 선택
3. 좌상단 기점의 점령 영역이 선택한 색으로 변하며, 인접한 동색 타일을 흡수해 영역 확장
4. 전체 그리드를 단색으로 만들면 클리어
5. 제한 이동 횟수 안에 완성해야 함 (기본 25회)

```
[시작]                [3턴 후]               [클리어]
R C G O P Y          C C G O P Y           C C C C C C
C G O P Y R    →     C C O P Y R    →  …   C C C C C C
G O P Y R C          G C P Y R C           C C C C C C
```

- 이동 횟수 초과 시 `--warn` 색으로 깜빡임

## 그래픽 재사용

| 리소스 | 재사용 방법 |
|--------|------------|
| `COLOR_PALETTE` 6색 | 타일 fill/glow 그대로 사용 |
| `ParticleSystem` | 색상 흡수 순간 파티클 |
| `FloatText` | 콤보 흡수 시 `+N` 팝업 |
| 배경 스캔라인 + 코너 글로우 | 무수정 재사용 |
| `sound.js` | 흡수음/클리어 팡파레 음표만 교체 |
| CSS 토큰 전체 | 무수정 |

신규 그래픽: **없음** (점령 경계 펄스는 Canvas `ctx.shadowBlur` 30줄)

## 개발 난이도

**2 / 5** — BFS flood-fill 함수 1개가 핵심 로직 전부.  
`GameLogic.applyInput()` 교체 → BFS 작성 → 색상 버튼 UI 추가.  
나머지(렌더러, 광고, 사운드)는 기존 코드 무수정 재사용.

**예상 프로토타입 기간: 3-5일**
