# Neon Drift — 게임 피치

> 밀어라. 같은 색이 만나면 합쳐진다. 한 색만 남을 때까지.

---

## 한 줄 소개

방향 버튼으로 전체 그리드를 밀어서 같은 색 블록을 합치고, 단 하나의 색으로 그리드를 지배하는 전략 슬라이드 퍼즐.

---

## 장르 / 레퍼런스

- 장르: 슬라이드 병합 퍼즐 (2048 + 테트리스 하이브리드)
- 레퍼런스: 2048, Threes!, Merged!
- 차별점: 숫자 대신 **색상 병합** — 색 계층 없이 동색만 합쳐짐. 목표는 최대 점수가 아닌 **단색 지배** 달성.

---

## 핵심 메커닉

1. 8×8 그리드에 3색 블록 배치로 시작
2. 방향 버튼(↑↓←→)으로 **전체 그리드를 한 방향으로 밀기**
3. 같은 색 블록 2개가 충돌하면 **합쳐져 크기 2 블록**으로 변환 (점수 획득)
4. 밀기 후 빈 칸에 새 블록 1개 자동 등장 (색상은 현재 그리드의 최다색 편향 랜덤)
5. 그리드가 꽉 차면 게임 오버
6. 연속 합치기(체인): 한 번 밀기로 복수 합치기 발생 시 배율 보너스

```
[밀기 전 →]          [밀기 후]
□ R □ R □           □ □ □ RR □    ← R+R 합치기
□ C C □ □    →      □ □ □ CC □    ← C+C 합치기
□ □ G □ G           □ □ □ □ GG   ← G+G 합치기
```

## 화면 구성

```
┌──────────────────────────┐
│  NEON DRIFT   LV.3       │
│  SCORE: 8,240  CHAIN: ×2 │
├──────────────────────────┤
│                          │
│   ┌──────────────┐       │
│   │  8×8 GRID   │       │  ← Canvas (슬라이드 트윈 애니메이션)
│   │  블록 슬라이드│       │
│   └──────────────┘       │
│                          │
├──────────────────────────┤
│      [↑]                 │
│  [←] [●] [→]            │  ← 방향 버튼 (네온 글로우 CSS 버튼)
│      [↓]                 │
└──────────────────────────┘
```

- 슬라이드 애니메이션: Canvas lerp 60fps, 50-80ms 트윈
- 합치기 순간: 파티클 폭발 + `FloatText` 점수
- 체인 달성: 화면 전체 글로우 플래시

## 그래픽 재사용

| 리소스 | 재사용 방법 |
|--------|------------|
| `COLOR_PALETTE` 6색 | 블록 색상 (레벨 진행에 따라 3→6색 확장) |
| `ParticleSystem` | 합치기 순간 파티클 |
| `FloatText` | 체인 배율 + 점수 팝업 |
| CSS 버튼 토큰 | 방향 버튼 4개 |
| 배경 + CSS 토큰 | 무수정 |
| `DIFFICULTIES` 패턴 | 색상 수 3→6 난이도 확장 이식 |

신규 그래픽: 방향 버튼 레이아웃 (CSS, 신규), 블록 크기별 숫자 표시 (합쳐진 블록 크기 = 2, 4, 8...)

## 개발 난이도

**3 / 5** (자체평가 2/5는 낙관적 — 실제 3/5)  
슬라이드 물리, 합치기 판정, 체인 계산, 트윈 애니메이션 동기화가 핵심 공수.  
`GameLogic` 버리고 `SlideGameLogic` 새로 작성 필요.  
`Array.shift/unshift`로 밀기 구현, 빈 셀 삽입으로 처리.

**예상 프로토타입 기간: 1-1.5주**

---

## 리텐션 훅

- 독창성 최고(5개 중 1위, 6.3): 테트리스+2048 복합 포지셔닝으로 마케팅 소구
- 색상 수 3→6 확장 난이도 곡선
- 체인 콤보 도파민 루프: 한 번 밀기로 연속 합치기 달성 쾌감

---

## 리스크

- **통제감 상실**: 매 밀기마다 전체 그리드가 움직이는 구조 — 원치 않는 합치기 발생 시 H5 캐주얼 유저 이탈
- **구현 복잡도 과소평가**: 트윈 타이밍 + 합치기 우선순위 + 체인 딜레이 조율이 예상보다 까다로움
- 보완책: 밀기 취소(Undo) 버튼 1회 기본 제공
