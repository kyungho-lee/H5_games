# Neon Line Surge — 게임 피치

> 블록을 놓아라. 단색 줄을 완성하라. 네온이 폭발한다.

---

## 한 줄 소개

색상이 고정된 블록 조각을 그리드에 배치해 **단색 행 또는 열**을 완성하면 클리어되는 전략 배치 퍼즐.

---

## 장르 / 레퍼런스

- 장르: 블록 배치 퍼즐 (1010! 변형)
- 레퍼런스: 1010!, Block Puzzle, Tetris (배치 개념만)
- 차별점: **단색 라인만 클리어** — 혼색 라인은 제거되지 않음. 이것이 핵심 전략 레이어.

---

## 핵심 메커닉

1. 10×10 그리드, 비어있는 상태로 시작
2. 화면 하단에 **3개의 블록 조각** 제시 (각 조각은 단일 색상 고정)
3. 원하는 위치에 드래그로 배치
4. **한 색으로만 채워진 행 또는 열**이 완성되면 폭발 제거 + 점수
5. 혼색 행/열은 완성되어도 제거 안 됨 → 전략적 배치 필요
6. 3개 조각 모두 배치하면 새 조각 세트 등장
7. 배치 공간이 없으면 게임 오버

```
그리드 예시:
■ ■ ■ ■ ■ ■ ■ ■ □ □  ← 마지막 2칸이 비어있어 클리어 안됨
□ ■ □ ■ □ □ □ □ □ □  ← 혼색, 클리어 불가
C C C C C C C C C C  ← 단색(Cyan) 완성 → 폭발!
```

## 화면 구성

```
┌──────────────────────────┐
│  NEON SURGE    SCORE     │
│              123,450     │
├──────────────────────────┤
│                          │
│   □□□□□□□□□□            │
│   □□□□□□□□□□            │  ← 10×10 Canvas 그리드
│   ■■■■■■■■■■  ← BOOM!  │     (완성된 단색 행: 글로우 폭발)
│   □□□□□□□□□□            │
│                          │
├──────────────────────────┤
│  [조각1] [조각2] [조각3]  │  ← 드래그 배치 영역
└──────────────────────────┘
```

- 단색 완성 직전 행/열: 해당 색 글로우 펄스 경고
- 드래그 중 배치 불가 위치: 빨간 오버레이
- 연속 클리어 콤보: `FloatText` 배율 팝업

## 그래픽 재사용

| 리소스 | 재사용 방법 |
|--------|------------|
| `COLOR_PALETTE` 6색 | 블록 조각 색상, 그리드 타일 fill/glow |
| `ParticleSystem` | 행/열 폭발 시 파티클 폭죽 |
| `FloatText` | 콤보 배율 팝업 (`×2`, `×3`) |
| `Renderer.drawTile()` | 그리드 타일 렌더링 그대로 |
| 배경 + CSS 토큰 | 무수정 |
| `sound.js` | 배치음/폭발음/콤보 팡파레 |

신규 그래픽: 조각 프리뷰 패널 (CSS div, `--panel` 토큰), 드래그 고스트 (Canvas globalAlpha)

## 광고 삽입 포인트

| 타이밍 | 유형 | 트리거 |
|--------|------|--------|
| 10라인 클리어마다 레벨업 | 인터스티셜 | `showInterstitial()` |
| 게임 오버 → 구출 옵션 | 리워드 | 그리드 꽉 차기 직전 랜덤 타일 3개 제거 |
| Last Chance (배치 공간 1개 남음) | 리워드 | 조각 1개 교체권 |

"그리드 꽉 차기 직전 구출" — 손실 회피 심리 피크에서 제시, 리워드 수락률 ↑

## 개발 난이도

**3 / 5** — 단색 행/열 감지는 O(n) 선형 스캔으로 단순하나,  
드래그 배치 UX (터치 좌표 → 그리드 셀 스냅, 고스트 프리뷰) 구현이 핵심 공수.  
`GameLogic` 대신 새 배치 엔진 작성 필요.

**예상 프로토타입 기간: 1-1.5주**

---

## 리텐션 훅

- 무한 엔드리스 + 점수 리더보드
- 색상 수 확장: 초반 3색 → 후반 6색 (SameGame `DIFFICULTIES` 패턴 이식)
- "완벽 클리어" 보너스: 연속 단색 2줄 동시 제거 시 배율
